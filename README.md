# Ticket-to-Hire 

**Assess engineers on real-world impact, not abstract algorithms.**

Ticket-to-Hire automates technical screening by converting job descriptions into realistic, AI-graded engineering challenges. It replaces LeetCode-style puzzles with JIRA-style tickets, simulating a candidate's actual "Day One" performance.

## Features

- **Job Description to Ticket Generation:** Instantly turns a job description (e.g., "Senior React Engineer") into a specific coding task.
- **AI Assessment and Grading:** Automatically evaluates submissions for code quality, security, and architectural patterns.
- **Smart Classification:** Classifies candidates as Junior, Mid, or Senior based on implementation depth.
- **Recruiter Dashboard:** A centralized, high-contrast dashboard to view submissions, summaries, and key strengths.

## Tech Stack

- **Framework:** [Next.js 14](https://nextjs.org/) (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS (Zinc/Dark Theme)
- **Backend & Auth:** Firebase (Firestore & Auth)
- **AI Engine:** Google Gemini API
